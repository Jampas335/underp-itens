Token do GitHub

Para publicar, editar ou deletar itens prontos, você precisa de um Personal Access Token do GitHub com permissão de escrita no repositório.

Repositório Jampas335/underp-itens

Para criar: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
Resource owner: Jampas335
Permissão necessária: Contents → Read and write no repositório acima.

Se o campo abrir preenchido com pontos, isso significa que já existe um token salvo nesta máquina. Cole outro somente se quiser substituir o atual.
